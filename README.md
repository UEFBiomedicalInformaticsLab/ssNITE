# ssNITE: Single-Sample Network Inference Through Trees in Ensemble

This repository contains the source codes for constructing single-sample networks using ssNITE. It also includes the scripts used to perform benchmarking of ssNITE against comparable methods.


## Publications

The following publications refer to this repository:

(1) [ssNITE: Single-Sample Network Inference Through Trees in Ensemble]()


## Running environment

The scripts have been developed and tested on R 4.3.3. The list of R packages required for running the scripts are listed in the file [Required_R_packages.csv](Environment/Required_R_packages.csv). The packages including all of its dependencies must be installed prior to execution of the scripts. The instructions for installing the CRAN packages can be found [here](https://www.rdocumentation.org/packages/utils/versions/3.6.2/topics/install.packages) and those for BioConductor packages can be found [here](https://www.bioconductor.org/install/).
