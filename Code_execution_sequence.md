# Code execution sequence


| Script Name | Description |
| :--- | :--- |
| **Data preparation** |
| [`Scripts/Build_network/STRING_PPI.R`](Scripts/Build_network/STRING_PPI.R) | Builds a protein-protein interaction network from the STRING database and extracts the largest connected component. |
| [`Scripts/Geneset_libraries/Prepare_geneset_from_databases.R`](Scripts/Geneset_libraries/Prepare_geneset_from_databases.R) | Prepares gene-disease associations by compiling data from numerous databases like OpenTargets, IntOGen, PharmGKB, and others. |
| [`Scripts/Geneset_libraries/KEGGpathway2Gene__human.R`](Scripts/Geneset_libraries/KEGGpathway2Gene__human.R) | Retrieves human KEGG pathway genes and maps them to Ensembl IDs. |
| [`Scripts/Omics_data_prepare/TCGABRCA_data_download.R`](Scripts/Omics_data_prepare/TCGABRCA_data_download.R) | Downloads TCGA-BRCA multi-omics data including RNA-Seq, clinical, proteomics, mutation, CNV, and methylation data. |
| [`Scripts/Omics_data_prepare/TCGABRCA_RNAseq_data_process.R`](Scripts/Omics_data_prepare/TCGABRCA_RNAseq_data_process.R) | Processes RNA-Seq count data by filtering low counts, applying Variance Stabilizing Transformation (VST), and removing outliers. |
| [`Scripts/AssociationsExtract_drug_interactors.R`](Scripts/AssociationsExtract_drug_interactors.R) | Extracts known human protein interactors and targets for small molecular drugs from DrugBank. |
| [`Scripts/AssociationsExtract_adverse_drug_drug_interactions.R`](Scripts/AssociationsExtract_adverse_drug_drug_interactions.R) | Analyzes the drug-drug interactions reported in DrugBank to identify combinations causing severe adverse drug reactions. |
| **Background gene Identification** |
| [`Scripts/SSN_background_genes/Extract_SSN_background_genes.R`](Scripts/SSN_background_genes/Extract_SSN_background_genes.R) | Short-lists a combined background gene set for network construction using cancer drugs, pathways, and oncogenes, and extracts their STRING sub-networks. |
| [`Scripts/SSN_background_genes/ClusterSamples_SSNbkgGene_expr_TGCABRCA.R`](Scripts/SSN_background_genes/ClusterSamples_SSNbkgGene_expr_TGCABRCA.R) | Checks the clustering of samples via PCA using only the expression levels of the extracted SSN background genes. |
| **Raw single-sample network (SSN) inference** |
| [`Scripts/Create_SSN/Create_ssNITEpy_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_ssNITEpy_SSN_TCGABRCA.R) | Prepares single-sample networks using the Python-based ssNITEpy random-forest method. |
| [`Scripts/Create_SSN/Create_BONOBO_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_BONOBO_SSN_TCGABRCA.R) | Prepares single-sample networks using the BONOBO algorithm. |
| [`Scripts/Create_SSN/Create_CSN_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_CSN_SSN_TCGABRCA.R) | Prepares cell-specific networks (CSN) via a MATLAB backend. |
| [`Scripts/Create_SSN/Create_LIONESS_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_LIONESS_SSN_TCGABRCA.R) | Prepares single-sample networks using the LIONESS algorithm. |
| [`Scripts/Create_SSN/Create_SWEET_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_SWEET_SSN_TCGABRCA.R) | Prepares sample-specific-weighted correlation networks (SWEET). |
| [`Scripts/Create_SSN/Create_BONOBOsparse_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_BONOBOsparse_SSN_TCGABRCA.R) | Prepares single-sample networks using the BONOBO algorithm in sparse mode. |
| [`Scripts/Create_SSN/Create_CSNsparse_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_CSNsparse_SSN_TCGABRCA.R) | Prepares cell-specific networks using the CSN algorithm in sparse mode. |
| [`Scripts/Create_SSN/Create_SWEETsparse_SSN_TCGABRCA.R`](Scripts/Create_SSN/Create_SWEETsparse_SSN_TCGABRCA.R) | Prepares sample-specific networks using the SWEET algorithm in sparse mode. |
| **Network edge refinement & sparsification** |
| [`Scripts/Edge_weight_stats/EdgeWtStat_SSN_TCGABRCA.R`](Scripts/Edge_weight_stats/EdgeWtStat_SSN_TCGABRCA.R) | Calculates and plots the edge weight statistics (distributions and densities) of the generated SSNs. |
| [`Scripts/Prune_SSN/Prune_SSN_TCGABRCA.R`](Scripts/Prune_SSN/Prune_SSN_TCGABRCA.R) | Prunes the full density SSNs by calculating thresholds based on scale-free topology fitting. |
| [`Scripts/Prune_SSN/Clean_sparse_SSN_TCGABRCA.R`](Scripts/Prune_SSN/Clean_sparse_SSN_TCGABRCA.R) | Cleans inherently sparse SSNs generated by sparse methods by removing zero-weight edges. |
| **Network topology analysis** |
| [`Scripts/Compute_SSN_centrality/Compute_prunedSSN_centrality_TCGABRCA.R`](Scripts/Compute_SSN_centrality/Compute_prunedSSN_centrality_TCGABRCA.R) | Computes pruned SSNs centralities (degree, closeness, betweenness, etc.) and performs PCA clustering. |
| [`Scripts/Compare_SSN/Compute_prunedSSN_similarity_TCGABRCA.R`](Scripts/Compare_SSN/Compute_prunedSSN_similarity_TCGABRCA.R) | Computes similarity metrics (Jaccard nodes/edges, Adjusted Rand Index) between the pruned SSNs. |
| [`Scripts/Corel_SSNtopol_vs_sampleInfo/Corel_ssnTopol_vs_sampleInfo_TCGABRCA.R`](Scripts/Corel_SSNtopol_vs_sampleInfo/Corel_ssnTopol_vs_sampleInfo_TCGABRCA.R) | Checks the correlation between SSN topological features and sample clinical information. |
| **Prediction of clinical traits** |
| [`Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/prepare_ML_stratifiedSamples_TCGABRCA.R`](Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/prepare_ML_stratifiedSamples_TCGABRCA.R) | Splits the TCGA-BRCA samples to generate stratified train and test cross-validation splits for ML analysis. |
| [`Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML_geneExpr_vs_sampleInfo_TCGABRCA.R`](Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML_geneExpr_vs_sampleInfo_TCGABRCA.R) | Trains ML models to predict TCGA-BRCA sample clinical traits directly from PCA-reduced gene expression data. |
| [`Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML__ssnTopol_vs_sampleInfo_TCGABRCA.R`](Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML__ssnTopol_vs_sampleInfo_TCGABRCA.R) | Trains ML models to predict TCGA-BRCA sample clinical traits using network-level topological properties. |
| [`Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML_ssnCentralityPC_vs_sampleInfo_TCGABRCA.R`](Scripts/ML_SSNtopol_vs_sampleInfo/TCGABRCA/ML_ssnCentralityPC_vs_sampleInfo_TCGABRCA.R) | Trains ML models to predict TCGA-BRCA sample clinical traits from principal components of node/edge centralities. |
| **Cross-method benchmarking & summary visualizations** |
| [`Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_compare_edge_density.R`](Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_compare_edge_density.R) | Calculates and plots comparative edge weight density distributions across all methods for publication summarization. |
| [`Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_compare_pruneSSN_topology.R`](Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_compare_pruneSSN_topology.R) | Compiles the pruned SSN topological parameters from different methods into comparative boxplot figures. |
| [`Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_prunedSSN_similarity.R`](Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_prunedSSN_similarity.R) | Summarizes the pairwise similarities between SSNs and checks their significance against clinical groups. |
| [`Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_summariseCorel_SSNtopol_vs_sampleInfo.R`](Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_summariseCorel_SSNtopol_vs_sampleInfo.R) | Summarizes the correlation between SSN topological properties and sample information into heatmaps and winner plots. |
| [`Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_summariseMLres_SSNtopol_vs_sampleInfo.R`](Scripts/Compare_X_methods/TCGABRCA/TCGABRCA_summariseMLres_SSNtopol_vs_sampleInfo.R) | Plots summary grids comparing machine learning prediction accuracies across network topologies, centralities, and gene expression. |
